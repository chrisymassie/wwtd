Travis simulator so you do not need to wait for the build.<br/>
Reads your .travis.yml and runs what travis would run.

Install
=======

```Bash
gem install wwtd
```

Usage
=====

```Bash
wwtd
Ruby ree with gemfiles/rails23.gemfile
....
Ruby 1.9.3 with gemfiles/rails23.gemfile
....
```
